# Devkit (CLI)

`@sublime-ui/devkit` is the `sublime` command line. It scaffolds code, generates
the cross-platform artifacts, and drives builds.

| Command | What it does |
| --- | --- |
| `sublime make:model <Name>` | Scaffold a Model. |
| `sublime make:component <Name>` | Scaffold a cross-platform component quartet. |
| `sublime theme:init` | Scaffold design tokens. |
| `sublime build:nav` | Compile storybooks into navigators + route types. |
| `sublime desktop:dev` / `desktop:build` | Run and package the desktop app (Electron Forge). |
| `sublime build` / `run` | Offline Android build + device run. |
